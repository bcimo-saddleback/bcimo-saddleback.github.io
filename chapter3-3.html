Chapter 3: Working with Text and Links

Section 3: Other Types of Links

Linking to an Email Address

You can create a link that opens the user’s default email client by using the >mailto: protocol in the href attribute.

<a href="mailto:example@example.com">Send an Email</a>

When the user clicks this link, it will open their default email application with the specified email address pre-filled.

Linking to a Phone Number

To create a link that allows users to call a phone number directly from a webpage (especially on mobile devices), you can use the tel: protocol.

<a href="tel:+1234567890">Call Us</a>

This will prompt the user to dial the specified number when clicked. It's important to include the country code (e.g., +1 for the United States) to ensure the number is dialed correctly, especially for international users.

File Download Links

You can link to files (like PDFs, images, or Word documents) so that users can download them. This is done in the same way as a regular link.

<a href="files/example.pdf" download>Download PDF</a>

The download attribute tells the browser to download the file instead of opening it in the browser.

Review and Summary

In this chapter, we have covered the following key concepts:

  • How to format text in HTML using headings, paragraphs, bold, italics, and lists.
  • How to create hyperlinks with the <a> tag, both to external websites and within the same page.
  • How to handle different types of links, such as email links, phone number links, and download links.

Remember that text formatting and links are foundational to building usable and accessible websites. Practice these techniques regularly to become comfortable with them.
