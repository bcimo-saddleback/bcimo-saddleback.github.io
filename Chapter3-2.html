Chapter 3: Working with Text and Links

Section 2: Creating Hyperlinks Using the <a> Tag

Hyperlinks are the backbone of the web, enabling users to navigate from one webpage to another or to other resources. In HTML, hyperlinks are created using the <a> tag.

Basic Hyperlinks

The <a> tag requires the href attribute, which specifies the destination of the link. A basic example looks like this:

<a href="https://www.example.com">Visit Example Website</a>

When a user clicks on this link, they will be directed to "https://www.example.com".

Opening Links in a New Tab

By default, links open in the same tab. If you want a link to open in a new tab, you can add the target="_blank" attribute:

<a href="https://www.example.com" target="_blank">Visit Example Website</a>

Descriptive Link Text

For accessibility reasons, always make sure your link text is descriptive and meaningful. Avoid generic link texts like "Click Here" or "Read More". Instead, describe what the link leads to, e.g., "Visit the Web Development Tutorial" or "Download the User Manual."

Adding Title Attribute

You can add a title attribute to links, which will display additional information when the user hovers over the link. While this can be useful, it should not be relied on as the sole way to convey important information.

<a href="https://www.example.com" title="Visit the Example Website">Visit Example Website</a>
