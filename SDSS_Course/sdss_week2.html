Astronomy Background

Detectors
CCD Architecture & Quantum Efficiency
A CCD is a silicon grid of pixels that converts photons to electrons via the photoelectric effect. Quantum efficiency (QE) — the fraction of photons actually detected — peaks around 70–90% for modern astronomical CCDs, far exceeding photographic plates (~5%) or the human eye (~1%). SDSS's 120-megapixel mosaic camera achieved QE >70% across the ugriz bands.
Detector Noise
Gain, Bias, Dark Current & Read Noise
Four noise sources shape CCD data: bias (a constant electronic offset), dark current (thermal electrons masquerading as signal), read noise (amplifier noise during readout), and Poisson/shot noise (statistical fluctuations in photon counts). Each is measured and removed during calibration.
Image Calibration
Bias, Dark & Flat-Field Frames
Raw CCD images must be calibrated before science use. A bias frame (zero-second exposure) removes the electronic offset; a dark frame removes thermal signal; a flat-field frame (uniformly illuminated screen) corrects pixel-to-pixel sensitivity variations. The SDSS pipeline does all three automatically.
Atmospheric Optics
Seeing & the Point Spread Function (PSF)
Atmospheric turbulence smears starlight into a blurry disk. "Seeing" quantifies this as the FWHM of the resulting PSF in arcseconds — typically 0.5–2″ at good sites. SDSS's Apache Point site achieves median seeing ~1.4″. The PSF shape also determines how well stars can be separated from neighboring galaxies.
Photometry
Aperture vs. PSF vs. Model Magnitudes
Different magnitude types suit different science goals. Aperture photometry sums flux within a fixed circle. PSF magnitudes fit a PSF model — ideal for stars. Petrosian magnitudes use an adaptive radius enclosing most galaxy light without excessive sky noise. Model magnitudes fit de Vaucouleurs + exponential profiles — SDSS's standard for galaxy colors.
Dust
Interstellar Extinction & Reddening
Interstellar dust absorbs and scatters blue light more than red, making distant objects appear both fainter (extinction) and redder (reddening). SDSS corrects for this using the Schlegel, Finkbeiner & Davis (1998) dust maps. Each PhotoObj row includes extinction_u/g/r/i/z columns encoding this correction.

The SDSS ugriz Filter System

Why five filters? A single exposure records total brightness; five filters sample how brightness changes with wavelength. Color indices like g−r and r−i encode stellar temperature, galaxy type, and redshift — making the ugriz system the backbone of all SDSS photometric classification.

SDSS PhotoObj: Key Columns

PhotoObj is the central imaging catalog — roughly 1.2 billion rows in DR17. Understanding which magnitude type to use, and how to apply flag cuts, is the single most important data skill this week.
Column Type When to Use It
psfMag_[ugriz] PSF magnitude Stars — best S/N for point sources; do NOT use for galaxies (underestimates flux)
modelMag_[ugriz] Model magnitude Galaxy colors — fits de Vaucouleurs + exponential profile; use for color indices
petroMag_[ugriz] Petrosian magnitude Galaxy total flux — aperture grows with galaxy size; luminosity function studies
fiberMag_[ugriz] Fiber magnitude Flux within 3″ fiber; matches spectroscopic targeting — use for fiber-matched photometry
extinction_[ugriz] Dust correction Subtract from observed magnitude to get dereddened magnitude: mag₀ = mag − extinction
petroRad_r Size (arcsec) Petrosian radius in r-band; defines galaxy size and morphology proxy
type Integer flag 3 = galaxy, 6 = star — primary morphological classifier; always filter on this
clean Boolean flag = 1 removes blended, saturated, edge, and bad astrometry objects; always apply for science
flags 64-bit bitmask Encodes 64 photometric status bits — deblended, saturated, interpolated, etc.
run, camcol, field Survey geometry Identifies which imaging run, camera column, and field the object falls in
The clean Flag
Always Use clean = 1
Setting clean = 1 in your SQL WHERE clause eliminates the most common data quality problems in a single step: blended detections, saturated pixels, objects on field edges, and failed astrometric fits. It is the equivalent of applying ~20 individual bitmask cuts simultaneously.
Drift-Scan Geometry
Runs, Camcols & Fields
SDSS imaged in six parallel "camera columns" (camcol 1–6) during each scan run. Each run covers the sky in sequential fields (~13.5′ × 9.8′). This geometry creates known stripes in the survey footprint visible in density maps — important to account for in clustering studies.

Working with SDSS FITS Images

SDSS imaging is stored as FITS files accessible via the DR17 image server. Below is a minimal Python workflow to fetch a corrected frame, measure the PSF, and run aperture photometry.

Python 3 · Jupyter Notebook week2_ccd_photometry.ipynb 
import numpy as np
import matplotlib.pyplot as plt
from astropy.io import fits
from astropy.stats import sigma_clipped_stats
from astropy.visualization import ZScaleInterval
from photutils.aperture import CircularAperture, aperture_photometry
from photutils.detection import DAOStarFinder

# ─── 1. Load a corrected SDSS frame (fpC file) ────────────────────────────
# URL pattern: https://dr17.sdss.org/sas/dr17/eboss/photo/redux/...
hdu = fits.open('frame-r-004192-4-0111.fits')
image  = hdu[0].data.astype(float)
header = hdu[0].header

gain   = header.get('GAIN', 4.72)    # electrons per ADU
sky    = header.get('SKY',  200.0)   # sky background (ADU)

# ─── 2. Estimate sky background ───────────────────────────────────────────
mean, median, std = sigma_clipped_stats(image, sigma=3.0)
image_sub = image - median            # subtract sky

# ─── 3. Visualise with ZScale stretch (standard in ds9) ───────────────────
interval = ZScaleInterval()
vmin, vmax = interval.get_limits(image_sub)

fig, ax = plt.subplots(figsize=(9, 9))
ax.imshow(image_sub, origin='lower', cmap='gray', vmin=vmin, vmax=vmax)
ax.set_title('SDSS r-band corrected frame — field 0111')
plt.tight_layout()

# ─── 4. Detect sources with DAOStarFinder ─────────────────────────────────
daofind = DAOStarFinder(fwhm=3.5, threshold=5.0 * std)
sources = daofind.find_stars(image_sub)
print(f"Found {len(sources)} sources")

# ─── 5. Aperture photometry ────────────────────────────────────────────────
positions  = np.array([sources['xcentroid'],
                          sources['ycentroid']]).T
apertures  = CircularAperture(positions, r=5.0)  # 5-pixel radius
phot_table = aperture_photometry(image_sub, apertures)

# ─── 6. Convert ADU counts to AB magnitude ────────────────────────────────
# SDSS zero-point: r-band aa ≈ -24.15 from the frame header
aa     = header.get('PHO_AA', -24.15)
kk     = header.get('PHO_KK', 0.10)   # atmospheric extinction coeff
airmass= header.get('AIRMASS', 1.1)

flux   = phot_table['aperture_sum'] * gain   # electrons
mag_inst = -2.5 * np.log10(flux)
mag_ab   = mag_inst - aa - kk * airmass      # calibrated AB mag

print(phot_table['xcenter','ycenter'], mag_ab[:10])
