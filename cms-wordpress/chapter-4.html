Themes, Widgets & Plugins

Customize the look of your site and extend its functionality.

WordPress Themes

A themeA WordPress theme controls the visual presentation of your website — colors, fonts, layouts, and page structure — without affecting your content. controls the visual design of your website. WordPress has two modern theme types:

Classic Themes use PHP template files, a traditional Customizer, and widget areas.

  • Customized via Appearance → Customize
  • Use widget areas (sidebar, footer)
  • Thousands available at WordPress.org
  • Examples: Astra, OceanWP, GeneratePress

Block Themes (Full Site Editing) allow you to edit every part of your site with the block editor.

  • Edit headers, footers, and templates visually
  • Customized via Appearance → Editor
  • Use Global Styles for site-wide design tokens
  • Examples: Twenty Twenty-Four, Kadence, Blocksy

Installing a Theme

1
Go to Appearance → Themes

In your admin dashboard, navigate to Appearance → Themes to see installed themes.

2
Click "Add New Theme"

Browse thousands of free themes from the official WordPress.org repository. Filter by Popular, Latest, or use the Feature Filter.

3
Preview, Install & Activate

Hover over a theme and click "Preview" to see it live, then "Install" to download it. Click "Activate" to make it your active theme.

Widgets

WidgetsWidgets are small, self-contained modules that add specific functionality to widget-ready areas of your WordPress site, like sidebars and footers. are modular content blocks placed in widget-ready areas (sidebars, footers, headers). In classic themes, go to Appearance → Widgets.

Common widgets include: Recent Posts, Search Bar, Categories, Archives, Tag Cloud, Calendar, Custom HTML, and Text.

ℹ️
Widgets in Block Themes

Block themes don't use traditional widgets. Instead, you add blocks directly to your template parts (header, footer, sidebar) using the Site Editor. All blocks function as "widgets" in this context.

Plugins

PluginsPlugins are packages of code that extend WordPress functionality. There are over 59,000 free plugins in the official WordPress plugin directory. add features to WordPress without editing any code. They range from contact forms to e-commerce to SEO tools.

Plugin Best Practices

Only install plugins from trusted sources. Too many plugins slow your site down and increase security risk. Deactivate AND delete plugins you no longer use — deactivating alone leaves files on the server.
