Chapter One

Hosting Options & Installing WordPress

Understand the hosting landscape and get WordPress running on your own server.

What is WordPress?

WordPress is the world's most popular Content Management System (CMS)A CMS is software that allows you to create, manage, and publish digital content without requiring advanced technical knowledge., powering over 43% of all websites on the internet. It began as a blogging platform in 2003 and has evolved into a flexible full-featured web publishing platform.

There are two distinct flavors you should know:

WordPress.com is a hosted platform — Automattic hosts everything for you.

  • ✅ No server management required
  • ✅ Free plan available
  • ⚠️ Limited plugin access on lower tiers
  • ⚠️ Less customization control
  • Best for: personal blogs, simple sites, beginners

WordPress.org is self-hosted — you download the software and run it on your own hosting.

  • ✅ Full control over everything
  • ✅ Install any theme or plugin
  • ⚠️ Requires web hosting & domain
  • ⚠️ You manage backups and updates
  • Best for: businesses, developers, serious bloggers

Installing WordPress — The Famous 5-Minute Install

1
Purchase Hosting & Domain

Choose a hosting provider (Bluehost, SiteGround, DreamHost). Register a domain name (yoursitename.com).

2
Create a MySQL Database

In your hosting control panel (cPanel), create a new database, username, and password. Note these down.

3
Download & Upload WordPress

Download from wordpress.org/download, then upload via FTP or your host's file manager to the public_html folder.

4
Run the Installer

Navigate to your domain in a browser. The WordPress setup wizard will guide you through connecting to your database.

5
Set Up Your Admin Account

Create a strong username, password, and enter your site title and email. Click "Install WordPress."

The wp-config.php File

During installation, WordPress creates a critical configuration file. Understanding it is essential:

wp-config.php
// Database configuration
define( 'DB_NAME', 'your_database_name' );
define( 'DB_USER', 'your_db_username' );
define( 'DB_PASSWORD', 'your_db_password' );
define( 'DB_HOST', 'localhost' );

// Security Keys (generated automatically)
define( 'AUTH_KEY', 'put your unique phrase here' );
define( 'SECURE_AUTH_KEY', 'put your unique phrase here' );
