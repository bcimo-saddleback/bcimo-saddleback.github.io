Content Creation & Management

Master the WordPress Block Editor and manage your media library.

The Block Editor (Gutenberg)

Since WordPress 5.0, the default editor is the Gutenberg Block EditorA visual page builder built into WordPress where content is created using individual "blocks" — each block is a discrete piece of content like a paragraph, image, or button.. Everything you create is made of blocks.

ℹ️
What is a Block?

A block is a self-contained content element. Common blocks include: Paragraph, Heading, Image, Gallery, Video, Quote, List, Button, Columns, and hundreds more via plugins.

Essential Block Types

Paragraph — Basic text. Heading (H1–H6) — Section titles. List — Bullet or numbered. Quote — Styled blockquotes. Code — Monospace formatted code. Table — Data tables with rows and columns.

Image — Single photos with alt text, captions, and links. Gallery — Multiple images in a grid. Video — Self-hosted or YouTube embeds. Audio — MP3 and other audio files. Cover — Image with text overlay.

Columns — Side-by-side content. Group — Wrap blocks together with shared styling. Spacer — Add whitespace. Separator — Horizontal dividers. Row — Horizontal block layouts.

WordPress auto-embeds content from YouTube, Twitter/X, Instagram, Spotify, TikTok, and many more. Just paste the URL on its own line, or use the dedicated Embed block.

Managing the Media Library

The Media Library (found under Media in the left sidebar) is where all uploaded files live — images, PDFs, videos, audio, and documents.

1
Uploading Media

Go to Media → Add New. Drag and drop files or click "Select Files." WordPress accepts JPEG, PNG, GIF, WebP, MP4, MP3, PDF, and more.

2
Adding Alt Text

Always add descriptive alt text to images for accessibility and SEO. Click any image in the library to edit its details.

3
Organizing with Folders (Plugin Required)

By default WordPress doesn't have media folders, but plugins like Filester or Enhanced Media Library add this functionality.
